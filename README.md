## Features

- Self-contained HTML files — no build step, no dependencies, just open in a browser
- Flip-card flashcards (20–25 per chapter)
- MCQ quizzes with instant feedback (15 per chapter)
- Syntax-highlighted SQL examples
- Comparison tables, tabbed panels, and interactive elements
- Dark theme, fully responsive

## Usage

Open `index.html` in a browser, or visit any `Ch{N}_*.html` file directly.
